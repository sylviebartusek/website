Fire Ant Frenzy
11/17/23
Java
Over the Fall 2023 semester, I have been coding a little 2D game engine from scratch in Java. This is the third and final game I made in my engine, called Fire Ant Frenzy.
This game features procedurally generated dungeons, smart enemies (ants) who use a decision tree to either flee from or swarm the main character (an anteater). the ants use A* pathfinding to swarm the enemy.
