Little Alchemy
10/11/23
Java
Over the Fall 2023 semester, I have been coding a little 2D game engine from scratch in Java. This is the second game I made in my engine, based off Little Alchemy.
Inspired by Little Alchemy, this game features collision detection, draggable components, and a viewport camera that can scale and pan. This game has 4 different combinations to unlock, playing mostly on compound words:
  • jelly + fish = jellyfish
  • ant + plate = anteater
  • jelly + plate = jello
  • jello + anteater = RARE SPECIAL JELLO ANTEATER
